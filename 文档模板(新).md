### 1.1 选择器

#### [强制] 当一个 `rule` 包含多个 `selector` 时，每个选择器声明必须独占一行。

示例：

    /* good */
    .post,
    .page,
    .comment {
        line-height: 1.5;
    }

    /* bad */
    .post, .page, .comment {
        line-height: 1.5;
    }
